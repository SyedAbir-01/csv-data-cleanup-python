# CSV Data Cleanup Automation (Python)

This project is a Python script that cleans and processes CSV files automatically.

## Features
- Removes duplicate rows
- Cleans text (trims spaces, normalizes casing)
- Preserves headers
- Outputs a clean CSV file

## Use Cases
- Cleaning customer lists
- Preparing data for reports
- Removing duplicates from exported data

## How to Run
1. Place your CSV file as `input.csv`
2. Run:
   python clean_csv.py
3. Get cleaned output in `output.csv`

## Technologies
- Python
- CSV module
